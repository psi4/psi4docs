:term:`PCG_CONVERGENCE <PCG_CONVERGENCE (DFOCC)>`
"""""""""""""""""""""""""""""""""""""""""""""""""

      Convergence criterion for residual vector of preconditioned conjugate gradient method. If this keyword is not set by the user, DFOCC will estimate and use a value required to achieve :term:`R_CONVERGENCE <R_CONVERGENCE (DFOCC)>` residual convergence. The listed default will be used for the default value of :term:`R_CONVERGENCE <R_CONVERGENCE (DFOCC)>` 

      * **Type**: :ref:`conv double <op_c_conv>`
      * **Default**: 1e-7
