:term:`CFOUR_SCF_CONV <CFOUR_SCF_CONV (CFOUR)>`
"""""""""""""""""""""""""""""""""""""""""""""""

      Specifies the convergence criterion for the HF-SCF equations. Equations are considered converged when the maximum change in density matrix elements is less than :math:`10^{-N}`. **Psi4 Interface:** Keyword subject to translation from value of :term:`D_CONVERGENCE <D_CONVERGENCE (SCF)>` unless set explicitly.

      * **Type**: integer
      * **Default**: 7
