:term:`CFOUR_GEO_MAXCYC <CFOUR_GEO_MAXCYC (CFOUR)>`
"""""""""""""""""""""""""""""""""""""""""""""""""""

      Specifies the maximum allowed number of geometry optimization cycles. **Psi4 Interface:** Geometry optimizations run through PSI (except in sandwich mode) use PSI's optimizer and so this keyword has no effect. Use :ref:`optking <apdx:optking>` keywords instead, particularly :term:`GEOM_MAXITER <GEOM_MAXITER (OPTKING)>` 

      * **Type**: integer
      * **Default**: 50
