:term:`CFOUR_INPUT_MRCC <CFOUR_INPUT_MRCC (CFOUR)>`
"""""""""""""""""""""""""""""""""""""""""""""""""""

      Specifies whether an input for mrcc is written (ON, =0) or not (OFF, =1) if :term:`CFOUR_CC_PROGRAM <CFOUR_CC_PROGRAM (CFOUR)>` =EXTERNAL has been specified.

      * **Type**: :ref:`boolean <op_c_boolean>`
      * **Default**: true
