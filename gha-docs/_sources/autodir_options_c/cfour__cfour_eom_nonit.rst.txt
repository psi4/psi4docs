:term:`CFOUR_EOM_NONIT <CFOUR_EOM_NONIT (CFOUR)>`
"""""""""""""""""""""""""""""""""""""""""""""""""

      Controls whether non-iterative triples corrections are applied after various types of EOM-CCSD calculation. Works with :term:`CFOUR_EXCITE <CFOUR_EXCITE (CFOUR)>` set to EOMIP, might work with EOMEE, certainly doesn't work with EOMEA. Use with great caution, preferably after having a few drinks.

      * **Type**: :ref:`boolean <op_c_boolean>`
      * **Default**: false
