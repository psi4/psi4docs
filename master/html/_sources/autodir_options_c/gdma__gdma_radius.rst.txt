:term:`GDMA_RADIUS <GDMA_RADIUS (GDMA)>`
""""""""""""""""""""""""""""""""""""""""

      The radii to be used, overriding the defaults. Specified as an array [ n1, r1, n2, r2, ... ] where n1,n2,n3... are atom type strings and r1,r2,r3 are radii in Angstrom.

      * **Type**: array
      * **Default**: No Default
