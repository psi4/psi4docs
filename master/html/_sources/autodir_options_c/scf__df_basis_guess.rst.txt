:term:`DF_BASIS_GUESS <DF_BASIS_GUESS (SCF)>`
"""""""""""""""""""""""""""""""""""""""""""""

      When :term:`BASIS_GUESS <BASIS_GUESS (SCF)>` is active, run the preliminary scf in density-fitted mode with this as fitting basis for the small basis set. A value of ``TRUE`` turns on density fitting with the default basis, otherwise the specified basis is used.

      * **Type**: string
      * **Possible Values**: :ref:`basis string <apdx:basisElement>`
      * **Default**: FALSE
