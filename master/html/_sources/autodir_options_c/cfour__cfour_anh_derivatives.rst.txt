:term:`CFOUR_ANH_DERIVATIVES <CFOUR_ANH_DERIVATIVES (CFOUR)>`
"""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""

      Specifies whether the anharmonic force field is calculated using analytic gradients (=FIRST) or analytic Hessians (=SECOND).

      * **Type**: string
      * **Possible Values**: FIRST, SECOND
      * **Default**: SECOND
