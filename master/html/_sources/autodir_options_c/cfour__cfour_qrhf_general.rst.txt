:term:`CFOUR_QRHF_GENERAL <CFOUR_QRHF_GENERAL (CFOUR)>`
"""""""""""""""""""""""""""""""""""""""""""""""""""""""

      The presence of this keyword specifies that a QRHF based CC calculation, or alternatively, an SCF calculation that uses the :term:`CFOUR_QRHFGUES <CFOUR_QRHFGUES (CFOUR)>` option, is to be performed.

      * **Type**: array
      * **Default**: No Default
