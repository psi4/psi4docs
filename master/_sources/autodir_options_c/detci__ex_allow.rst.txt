:term:`EX_ALLOW <EX_ALLOW (DETCI)>`
"""""""""""""""""""""""""""""""""""

      An array of length :term:`EX_LEVEL <EX_LEVEL (DETCI)>` specifying whether each excitation type (S,D,T, etc.) is allowed (1 is allowed, 0 is disallowed). Used to specify non-standard CI spaces such as CIST. 

      * **Type**: array
      * **Default**: No Default
