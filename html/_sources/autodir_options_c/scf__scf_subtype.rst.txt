:term:`SCF_SUBTYPE <SCF_SUBTYPE (SCF)>`
"""""""""""""""""""""""""""""""""""""""

      For certain :term:`SCF_TYPE <SCF_TYPE (GLOBALS)>` algorithms that have internal sub-algorithms depending on available memory or other hardware constraints, allow the best sub-algorithm for the molecule and conditions (``AUTO`` ; usual mode) or forcibly select a sub-algorithm (usually only for debugging or profiling). Presently, ``SCF_TYPE=MEM_DF`` and ``SCF_TYPE=DISK_DF`` can have ``INCORE`` 	 or ``OUT_OF_CORE`` selected. (This also applies for ``SCF_TYPE=DF``.) 

      * **Type**: string
      * **Possible Values**: AUTO, INCORE, OUT\_OF\_CORE
      * **Default**: AUTO
