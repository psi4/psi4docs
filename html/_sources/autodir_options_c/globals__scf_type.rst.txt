"""""""""""""""""""""""""""""""""""""

      What algorithm to use for the SCF computation. See Table :ref:`SCF Convergence & Algorithm <table:conv_scf>` for default algorithm for different calculation types.

      * **Type**: string
      * **Possible Values**: DIRECT, DF, MEM\_DF, DISK\_DF, PK, OUT\_OF\_CORE, CD, GTFOCK
