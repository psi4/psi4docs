:term:`GRADIENT_WRITE <GRADIENT_WRITE (FINDIF)>`
""""""""""""""""""""""""""""""""""""""""""""""""

      Do write a gradient output file? If so, the filename will end in .grad, and the prefix is determined by :term:`WRITER_FILE_LABEL <WRITER_FILE_LABEL (GLOBALS)>` (if set), or else by the name of the output file plus the name of the current molecule.

      * **Type**: :ref:`boolean <op_c_boolean>`
      * **Default**: false
