:term:`DF_BASIS_ELST <DF_BASIS_ELST (SAPT)>`
""""""""""""""""""""""""""""""""""""""""""""

      Auxiliary basis set for SAPT Elst10 and Exch10 density fitting computations, may be important if heavier elements are involved. :ref:`Defaults <apdx:basisFamily>` to a JKFIT basis. Previous to v1.6, defaulted to :term:`DF_BASIS_SAPT <DF_BASIS_SAPT (SAPT)>`  See :ref:`fitting notes <sec:saptfitA>` .

      * **Type**: string
      * **Possible Values**: :ref:`basis string <apdx:basisElement>`
      * **Default**: No Default
