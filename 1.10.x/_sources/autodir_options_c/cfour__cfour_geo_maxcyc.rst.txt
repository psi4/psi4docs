:term:`CFOUR_GEO_MAXCYC <CFOUR_GEO_MAXCYC (CFOUR)>`
"""""""""""""""""""""""""""""""""""""""""""""""""""

      Specifies the maximum allowed number of geometry optimization cycles. **Psi4 Interface:** Geometry optimizations run through PSI (except in sandwich mode) use PSI's optimizer and so this keyword has no effect. Use :ref:`optking <apdx:optking>` keywords instead, particularly :py:attr:`~optking.v1.optparams.OptParams.geom_maxiter`.

      * **Type**: integer
      * **Default**: 50
