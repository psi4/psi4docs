:term:`SAPT_DFT_GRAC_BASIS <SAPT_DFT_GRAC_BASIS (SAPT)>`
""""""""""""""""""""""""""""""""""""""""""""""""""""""""

      To ensure that the GRAC shift is computed with a sufficiently large basis set, the user can specify a larger basis set for the GRAC calculation, which can be different from the basis set used for the SAPT(DFT) calculation.

      * **Type**: string
      * **Default**: AUTO
