:term:`G_CONVERGENCE <G_CONVERGENCE (OPTKING)>`
"""""""""""""""""""""""""""""""""""""""""""""""

      Set of optimization criteria. Specification of any MAX_*_G_CONVERGENCE or RMS_*_G_CONVERGENCE options will append to overwrite the criteria set here unless :term:`FLEXIBLE_G_CONVERGENCE <FLEXIBLE_G_CONVERGENCE (OPTKING)>` is also on. See Table :ref:`Geometry Convergence <table:optkingconv>` for details.

      * **Type**: string
      * **Possible Values**: QCHEM, MOLPRO, GAU, GAU\_LOOSE, GAU\_TIGHT, INTERFRAG\_TIGHT, GAU\_VERYTIGHT, TURBOMOLE, CFOUR, NWCHEM\_LOOSE
      * **Default**: QCHEM
