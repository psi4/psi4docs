:term:`E_CONVERGENCE <E_CONVERGENCE (FNOCC)>`
"""""""""""""""""""""""""""""""""""""""""""""

      Convergence criterion for CC energy. See Table :ref:`Post-SCF Convergence <table:conv_corl>` for default convergence criteria for different calculation types. Note that convergence is met only when :term:`E_CONVERGENCE <E_CONVERGENCE (FNOCC)>` and :term:`R_CONVERGENCE <R_CONVERGENCE (FNOCC)>` are satisfied.

      * **Type**: :ref:`conv double <op_c_conv>`
      * **Default**: 1.0e-6
