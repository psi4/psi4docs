:term:`D_CONVERGENCE <D_CONVERGENCE (SCF)>`
"""""""""""""""""""""""""""""""""""""""""""

      Convergence criterion for SCF density, defined as the RMS or maximum absolute value of the orbital gradient. See Table :ref:`SCF Convergence & Algorithm <table:conv_scf>` for default convergence criteria for different calculation types. **Cfour Interface:** Keyword translates into :term:`CFOUR_SCF_CONV <CFOUR_SCF_CONV (CFOUR)>` 

      * **Type**: :ref:`conv double <op_c_conv>`
      * **Default**: 1e-6
