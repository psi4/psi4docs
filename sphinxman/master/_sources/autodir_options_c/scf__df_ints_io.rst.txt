:term:`DF_INTS_IO <DF_INTS_IO (SCF)>`
"""""""""""""""""""""""""""""""""""""

      IO caching for CP corrections, etc. Changing this selects Disk_DF over Mem_DF. Note that setting this forces DiskDFJK when SCF_TYPE=DF. 

      * **Type**: string
      * **Possible Values**: NONE, SAVE, LOAD
      * **Default**: NONE
