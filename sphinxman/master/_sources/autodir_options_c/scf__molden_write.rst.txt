:term:`MOLDEN_WRITE <MOLDEN_WRITE (SCF)>`
"""""""""""""""""""""""""""""""""""""""""

      Do write a MOLDEN output file? If so, the filename will end in .molden, and the prefix is determined by :term:`WRITER_FILE_LABEL <WRITER_FILE_LABEL (GLOBALS)>` (if set), or else by the name of the output file plus the name of the current molecule.

      * **Type**: :ref:`boolean <op_c_boolean>`
      * **Default**: false
