:term:`NORMAL_MODES_WRITE <NORMAL_MODES_WRITE (FINDIF)>`
""""""""""""""""""""""""""""""""""""""""""""""""""""""""

      Do write a file containing the normal modes in Molden format? If so, the filename will end in .molden_normal_modes, and the prefix is determined by :term:`WRITER_FILE_LABEL <WRITER_FILE_LABEL (GLOBALS)>` (if set), or else by the name of the output file plus the name of the current molecule.

      * **Type**: :ref:`boolean <op_c_boolean>`
      * **Default**: false
