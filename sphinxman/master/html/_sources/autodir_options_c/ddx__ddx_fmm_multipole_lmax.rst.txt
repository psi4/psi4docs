:term:`DDX_FMM_MULTIPOLE_LMAX <DDX_FMM_MULTIPOLE_LMAX (DDX)>`
"""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""

      Maximal degree of multipole spherical harmonics (far-field FMM interactions). Using the same value as :term:`DDX_LMAX <DDX_LMAX (DDX)>` is recommended and done by default.

      * **Type**: integer
      * **Default**: 9
