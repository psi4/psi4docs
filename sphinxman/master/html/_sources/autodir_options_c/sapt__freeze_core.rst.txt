:term:`FREEZE_CORE <FREEZE_CORE (SAPT)>`
""""""""""""""""""""""""""""""""""""""""

      The scope of core orbitals to freeze in evaluation of SAPT :math:`E_{disp}^{(20)}` and :math:`E_{exch-disp}^{(20)}` terms. Recommended true for all SAPT computations

      * **Type**: string
      * **Possible Values**: FALSE, TRUE
      * **Default**: FALSE
