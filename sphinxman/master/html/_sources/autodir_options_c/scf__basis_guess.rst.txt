:term:`BASIS_GUESS <BASIS_GUESS (SCF)>`
"""""""""""""""""""""""""""""""""""""""

      Accelerate convergence by performing a preliminary SCF with this small basis set followed by projection into the full target basis. A value of ``TRUE`` turns on projection using the :ref:`Defaults <apdx:basisFamily>` small basis set 3-21G, pcseg-0, or def2-SV(P).

      * **Type**: string
      * **Default**: FALSE
