:term:`OCC_PERCENTAGE <OCC_PERCENTAGE (FNOCC)>`
"""""""""""""""""""""""""""""""""""""""""""""""

      Cutoff for occupation of MP2 virtual NOs in FNO-QCISD/CCSD(T). The number of virtual NOs is chosen so the occupation of the truncated virtual space is :term:`OCC_PERCENTAGE <OCC_PERCENTAGE (FNOCC)>` percent of occupation of the original MP2 virtual space. This option is only used if :term:`NAT_ORBS <NAT_ORBS (FNOCC)>` = true. This keyword overrides :term:`OCC_TOLERANCE <OCC_TOLERANCE (FNOCC)>` 

      * **Type**: double
      * **Default**: 99.0
