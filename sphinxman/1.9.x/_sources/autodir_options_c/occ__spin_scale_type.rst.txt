:term:`SPIN_SCALE_TYPE <SPIN_SCALE_TYPE (OCC)>`
"""""""""""""""""""""""""""""""""""""""""""""""

      Controls the spin scaling set to current energy. This is set by Psi internally. 

      * **Type**: string
      * **Possible Values**: NONE, CUSTOM, SCS, SCSN, SCSVDW, SOS, SOSPI
      * **Default**: NONE
