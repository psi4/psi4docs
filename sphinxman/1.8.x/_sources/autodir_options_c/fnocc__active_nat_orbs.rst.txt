:term:`ACTIVE_NAT_ORBS <ACTIVE_NAT_ORBS (FNOCC)>`
"""""""""""""""""""""""""""""""""""""""""""""""""

      An array containing the number of virtual natural orbitals per irrep (in Cotton order) so a user can specify the number of retained natural orbitals rather than determining them with :term:`OCC_TOLERANCE <OCC_TOLERANCE (FNOCC)>`  This keyword overrides :term:`OCC_TOLERANCE <OCC_TOLERANCE (FNOCC)>` and :term:`OCC_PERCENTAGE <OCC_PERCENTAGE (FNOCC)>` 

      * **Type**: array
      * **Default**: No Default
