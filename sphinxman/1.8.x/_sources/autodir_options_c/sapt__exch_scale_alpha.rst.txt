:term:`EXCH_SCALE_ALPHA <EXCH_SCALE_ALPHA (SAPT)>`
""""""""""""""""""""""""""""""""""""""""""""""""""

      Whether or not to perform exchange scaling for SAPT exchange components. Default is false, i.e. no scaling. If set to true, performs scaling with :math:`Exch10 / Exch10(S^2)`. If set to a value :math:`\alpha`, performs scaling with :math:`(Exch10 / Exch10(S^2))^{\alpha}`.

      * **Type**: string
      * **Default**: FALSE
