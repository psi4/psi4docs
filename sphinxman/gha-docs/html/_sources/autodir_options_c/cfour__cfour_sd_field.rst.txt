:term:`CFOUR_SD_FIELD <CFOUR_SD_FIELD (CFOUR)>`
"""""""""""""""""""""""""""""""""""""""""""""""

      Specifies the strength of a spin-dipole perturbation as required for finite-field calculations of the SD contributions to indirect spin-spin coupling constants. The value must be specified as an integer and the SD strength used by the program will be the value of the keyword :math:`\times 10^{-6}`. (Default : 0, currently not implemented)

      * **Type**: integer
      * **Default**: 0
