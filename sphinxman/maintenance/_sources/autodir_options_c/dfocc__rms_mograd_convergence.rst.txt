:term:`RMS_MOGRAD_CONVERGENCE <RMS_MOGRAD_CONVERGENCE (DFOCC)>`
"""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""

      Convergence criterion for RMS orbital gradient. If this keyword is not set by the user, DFOCC will estimate and use a value required to achieve the desired :term:`E_CONVERGENCE <E_CONVERGENCE (DFOCC)>`  The listed default will be used for the default value of :term:`E_CONVERGENCE <E_CONVERGENCE (DFOCC)>` 

      * **Type**: :ref:`conv double <op_c_conv>`
      * **Default**: 1e-4
