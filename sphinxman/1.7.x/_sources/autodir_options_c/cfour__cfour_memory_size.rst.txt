:term:`CFOUR_MEMORY_SIZE <CFOUR_MEMORY_SIZE (CFOUR)>`
"""""""""""""""""""""""""""""""""""""""""""""""""""""

      Specifies the amount of core memory used in integer words (default) or in the units specified via the keyword :term:`CFOUR_MEM_UNIT <CFOUR_MEM_UNIT (CFOUR)>`  Default: 100 000 000 (approximately 381 or 762 MB for 32 or 64 bit machines, respectively). **Psi4 Interface:** Keyword set in MB from memory input command when given.

      * **Type**: integer
      * **Default**: 100000000
