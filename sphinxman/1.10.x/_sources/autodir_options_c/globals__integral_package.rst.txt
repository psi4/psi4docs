:term:`INTEGRAL_PACKAGE <INTEGRAL_PACKAGE (GLOBALS)>`
"""""""""""""""""""""""""""""""""""""""""""""""""""""

      Integral package to use. If compiled with Simint support, change this option to use them; LibInt2 is used otherwise.

      * **Type**: string
      * **Possible Values**: LIBINT2, SIMINT
      * **Default**: LIBINT2
