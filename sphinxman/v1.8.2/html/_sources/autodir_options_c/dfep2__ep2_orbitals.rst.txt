:term:`EP2_ORBITALS <EP2_ORBITALS (DFEP2)>`
"""""""""""""""""""""""""""""""""""""""""""

      Explicitly pick orbitals to use in the EP2 method, overrides :term:`EP2_NUM_IP <EP2_NUM_IP (DFEP2)>` and :term:`EP2_NUM_EA <EP2_NUM_EA (DFEP2)>` options. Input array should be [[orb1, orb2], [], ...] for each irrep.

      * **Type**: array
      * **Default**: No Default
